### 财富自由之路

财富自由的本质不是金钱，而是"不再为生活必需而出售自己的时间"——即时间自主权。全书确立"注意力 > 时间 > 金钱"的价值观排序，带你审视自己"出售时间"的个人商业模式，识别消耗注意力的"人生三大坑"，并把懂得的道理真正践行、内化。

### 财富的真相

钱到底从哪来？从第一性原理推演：财富唯一正当的来源是"生产"，而把钱变成财富的唯一手段是足够久的"积累"。最本质的生产资料是知识和时间，二者都源于自身——所以人人都能白手起家。投资与投机的分界，就在于是否把"时间"纳入生产资料。

### 让时间陪你慢慢变富

面向被"投资"吓退的普通人，这本书只给一个答案：定投——长期、定期、定额、选对标的。同样的价格曲线，定投者走出"微笑曲线"，追涨杀跌者走出"猥琐曲线"。它教你"主动地被动"，放下与趋势无关的鸡毛蒜皮，让钱不停替你赚钱。

### 思考的真相

很多人上了多年学，却依然不会思考。这本书把"如何思考"拆到最基础：只有定义、分类、比较、因果四个要素，核心是概念。任何决策，都是这四者按"定义 → 分类 → 比较 → 因果"的顺序组合应用；再复杂的问题，也能拆成这些简单要素逐一突破。

### 专注的真相

这个时代最大的苦恼是无法专注。先祛魅：多巴胺不是"快乐物质"，而是驱使你不停刷手机的"冲动物质"。注意力是最稀缺的资源，"有意义的专注要大于二十五分钟"，而衡量一切的标尺只有一把——"值吗？"值不值你一生的万分之一。真正的专注，扎根于价值观。

### 韭菜的自我修养

"韭菜"到底是谁？在本质并非零和游戏的市场里，却以为自己在玩零和游戏的交易者。因为交易所"抽水"，交易越频繁越接近零和，唯一赢家只有交易所——翻身之路只有一条：降低交易频次。看"周期"而非只盯"趋势"，靠群体情绪判断牛熊拐点。
