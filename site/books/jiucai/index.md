# 韭菜的自我修养

**降低交易频次，看懂周期**

“韭菜”到底是谁？李笑来的定义是：在本质并非零和游戏的市场里，却以为自己在玩零和游戏的交易者。

因为交易所“抽水”，交易越频繁越接近零和，唯一赢家只有交易所——翻身之路只有一条：降低交易频次。他还教你看“周期”而非只盯“趋势”，靠群体情绪判断牛熊拐点。
