# 财富的真相

**财富唯一正当的来源是生产**

钱到底从哪来？李笑来用第一性原理推演：财富唯一正当的来源是“生产”，而把钱变成财富的唯一手段是足够久的“积累”。

最本质的生产资料是知识和时间，二者都源于自身——所以人人都能白手起家。投资与投机的分界，就在于是否把“时间”纳入生产资料。
