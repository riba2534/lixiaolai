# 思考的真相

**思考只有四个要素**

很多人上了多年学，却依然不会思考。这本书把“如何思考”拆到最基础：只有定义、分类、比较、因果四个要素，核心是概念。

任何决策，都是这四者按“定义 → 分类 → 比较 → 因果”的顺序组合应用；再复杂的问题，也能拆成这些简单要素逐一突破。
