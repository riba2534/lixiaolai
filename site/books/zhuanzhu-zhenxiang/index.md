# 专注的真相

**注意力是最稀缺的资源**

这个时代最大的苦恼是无法专注。李笑来先祛魅：多巴胺不是“快乐物质”，而是驱使你不停刷手机的“冲动物质”。

他把注意力称作最稀缺的资源，指出“有意义的专注要大于二十五分钟”，并给出一把标尺——“值吗？”——值不值你一生的万分之一。真正的专注，扎根于价值观。
